# quarto-typst-header

A repo to document a less-than-desireable behavior of trying to set a
header in typst output.

``` r
quarto::quarto_version()
#> [1] '1.4.553'
```

``` r
system2("typst", "--version", stdout = TRUE)
#> [1] "typst 0.11.0"
```
